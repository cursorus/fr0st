# Fr0st

> ⚠️ **BETA SOFTWARE — USE AT YOUR OWN RISK.** Fr0st modifies SpringBoard at runtime using a kernel-level exploit. It may cause crashes, SpringBoard resets, or unexpected behavior. No guarantees. You’ve been warned.

**Fr0st** is a free, open-source iOS tweak runner — a fork of [zeroxjf/cyanide](https://github.com/zeroxjf/cyanide), built on top of the DarkSword kernel r/w primitive.

Everything is free. Everything is open. No Patreon. No gating. No bullshit.

-----

## What’s inside

- **Rounded Icons** — smooth corner radius applied to every home screen icon, no theme required
- **Watch Layout** — Apple Watch-style SpringBoard grid with circular icons and compact spacing
- **Gravity** — physics-based icon movement with accelerometer steering, collisions, and bounce (fixed)
- Everything from the original Cyanide: SBCustomizer, Powercuff, StatBar, and more

-----

## Requirements

- iPhone with A12–A18 chip
- iOS 18.x or iOS 26.x
- SideStore, TrollStore, or any sideloading tool

-----

## Install

1. Download the latest `.ipa` from [Releases](../../releases)
1. Install via SideStore or TrollStore
1. Open Fr0st and apply tweaks

-----

## Build from source

Fr0st builds with Xcode on macOS, or automatically via GitHub Actions on every push.

```bash
git clone https://github.com/YOUR_USERNAME/fr0st
cd fr0st
open Cyanide.xcodeproj
```
